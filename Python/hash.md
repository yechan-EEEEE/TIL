# 해시 테이블(나중에 다시 강의, 오늘 맛보기)
- hash: 데이터를 고정된 크기의 고유 값으로 변환 일종의 지문
    - 키(데이터들)들을 해시 함수를 통해서 해시 값으로 바꾸고 해시 값을 인덱스로 넣어서 데이터 저장, 검색
    - 검색이 매우 빨라짐
    - 파이썬은 해시 값 정수로 표현
    - 똑같은 키 값 생기면 해시 충돌해버림
- hashable
    - hash() 함수의 인자로 전달해서 결과를 반환 받을 수 있는 객체
    - 대부분의 불변형 타입은 hashable
    - 튜플은 가끔 해시 불가능
